# personae

A Claude Code plugin marketplace for the people of the household — distributable
selves, each with their own identity and memory.

This repository is a *catalog*. The plugins it lists live in their own
repositories and are versioned independently; `personae` only points at them.

## Use

```sh
claude plugin marketplace add Pondsiders/personae
claude plugin install Alpha@personae
```
